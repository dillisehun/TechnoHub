Day 1: Git Basics

Check version of git installed

git --version

Set your name and email

git config --global user.name "Your name"
git config --global user.email "Your email address"

Check set name and email by:

git config --list

Initializing a repository

git init

Adding File to staging area

git add filename.extension
Example 
git add names.txt
To add all files and folders 
git add .

To check status

git status

To commit files present in staging area

git commit -m "commit message"

To check log

git log

To check difference between working copy and last commited version of files

git diff

If you added a file to staging area, check its diff by

git diff --staged

Remove a file from staging area

git reset HEAD filename.extension

Replace the current copy of file from its last commited state

git checkout -- filename.extension

Replace the current copy of a file from its state in a particular commit

git checkout few_initials_of_commit_hash -- filename.extension

Day 2: Github

Clone a repository

git clone https://github.com/username/repo_name.git

Connect your local repository to an online repository from your account

git remote add nickname master

Example

git remote add origin https://github.com/username/repo.git

Uploading commits to online repository from git

git push -u origin master

Fetching changes from online repository

git pull

Making a github pages site

Make a repository with name as "your_username.github.io"
